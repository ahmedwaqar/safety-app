Input guidance

Choosing appropriate values

Start with architecture

Import PlantUML components before creating component-linked FMEA, FMEDA, quantitative, or requirement records. Use stable aliases such as SCAN or PLC.

FMEA ratings

Severity, occurrence, and detection use integer values from 1 to 10. Document your organization-specific rating scale and apply it consistently.

AMR SIL risk graph

Select C/F/P/W values from documented operating-zone evidence. Treat the resulting SIL as a target estimate for competent review, not an automatic compliance decision.

Failure rates

Enter finite non-negative rates in failures per hour. Fractions, diagnostic coverage, and beta factors must be between 0 and 1. Proof-test intervals are positive hours.

FMEDA symbols

Use names such as lambda_scanner. Expressions accept constants, numbers, parentheses, and +, -, *, /.

Portable JSON

Use Open JSON and the download button to move complete projects between browsers and operating systems.
